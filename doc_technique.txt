Générateur de poèmes en base algorithmique.

1 - inputs utilisateur :

    multiples variable affectant la génération du poème:

    structure -> type de poeme ex: sonnet, aiku, tercet ; type de donnée -> string
    syllabes -> nombre de syllabes par vers ; type de donnée -> integer
    type rimes -> type de rimes ex: ABAB(croisés, suivies) ; type de donnée -> list
    phonétique rimes = phonétique du poeme forme : "ou","é" ; type de donnée -> tuple
    theme = thematique du poeme ex: "nature" ; type de données -> string
